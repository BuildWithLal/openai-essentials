#### OpenAI Basics using Python

* This repo includes basics of OpenAI using Python and OpenAI API
* It covers interacting with OpenAI `GPT-3.5 Turbo`, `GPT-4o`, `TTS`, `Whisper`, `DALL-E`, `Fine-Tuning` models.

#### Tech Stack

* OpenAI
* python-dotenv
* notebook
